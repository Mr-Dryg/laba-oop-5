# Лаба № 5. Вариант 10
## Задание:

### 1. Реализовать наследник std::pmr::memory_resource.
Наследник реализует стратегию работы с памятью согласно варианту 4:
* Динамическое выделение памяти: для каждого объекта выделяется блок памяти на куче, информация о выделенных блоках сохраняется в std::list.

### 2. Реализовать шаблонный контейнер "Очередь".
Очередь использует созданный memory_resource через шаблон std::pmr::polymorphic_allocator.

### 3. Реализовать итератор к созданному контейнеру.
Итератор должен соответствовать std::forward_iterator_tag.


## О программе
### 1. Сборка
Запустить devcontainer и собрать с помощью CMake
### 2. Тестирование
Запуск тестирования
```
ctest
```
### 3. Запуск приложения
```
./laba_exe
```
